Changing a python script to exe:

pyinstaller <python file name>
	- goes in 'dist' folder
	
WITHOUT WINDOW OPENING

pyinstaller -w <python file name>

ONE EXE no other files

pyinstaller -F <python file name>

ADDING AN ICON - must be ico file

pyinstaller -i <icon file> <python file name>


CAN AMALGAMATE ALL OPTIONS TOGETHER
